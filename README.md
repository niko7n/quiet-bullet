# quiet-bullet
Sorry about the name. This is a minimal [SilverBullet](https://silverbullet.md/) theme that favors a quieter, easier-on-the-eyes aesthetic. The `space-style` blocks have been spread across multiple pages for a more sane development.

There are still several known issues and open TODOs.
